Phishing Attack

This project simulates phishing attacks to teach users how to identify and avoid online security threats. It’s strictly educational, aimed at raising awareness and improving online safety practices.

Project Details

This project was developed using PHP for the back-end and MySQL for database management. The goal was to create a simulation of phishing attacks to help users understand how phishing attempts can look and how they can avoid falling victim to them.

We used PHP for server-side scripting to manage the user interactions and simulate different phishing scenarios. The data regarding phishing examples was stored in a MySQL database, and PHP was used to retrieve this information dynamically.

Users can learn to identify phishing emails and websites through the simulations. The project also provides educational resources and tips on how to stay safe online.
